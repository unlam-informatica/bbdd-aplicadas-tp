# datos_origen

Archivos externos a importar (Entrega 6 / sección II.F del `tp.pdf`).

Colocar aquí los datasets **tal cual se descargan** (no deben modificarse con
herramientas externas antes de importarlos). Los SP de importación
([`../scripts/03_importacion/01_sp_importacion.sql`](../scripts/03_importacion/01_sp_importacion.sql))
reciben el nombre del archivo como parámetro.

## Requisitos

- Al menos **3 datasets** en al menos **2 formatos distintos** (CSV, XML, etc.).
- Fuentes gubernamentales / ONGs, por ejemplo:
  - https://www.untourism.int/
  - https://www.protectedplanet.net/en
  - https://www.ign.gob.ar/
  - https://datos.gob.ar/
  - https://datos.yvera.gob.ar/
- Documentar (en el informe) el origen de cada dataset.

> Nota: según `.gitignore`, los `*.csv`/`*.xlsx` grandes podrían no versionarse.
> Verificar antes de depender de que estén commiteados.
